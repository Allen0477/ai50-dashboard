信号灯算法：综合分 = 资金面30% + 估值面(扣非)25% + 技术面25% + 动态情绪20%（缺失维度按权重归一化，综合分列下方%为数据完整度）。
硬规则：①一次性损益预警 + 资金净流出 → 最多“减仓”；②技术破位 + 资金净流出 → “清仓”。阈值：≥68加仓 / 50–68持有 / 35–50减仓 / <35清仓。
⚠ 本看板为量化辅助，非投资建议。估值优先扣非口径以避开“伪低估”陷阱。资金面/技术面需 refresh_data.py 刷新当日数据后才完整。
